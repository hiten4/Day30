# Logistic Regression on SUV Dataset

This project is part of the AI-ML & Agentic AI Engineering program. It focuses on building an end-to-end machine learning pipeline using Logistic Regression on the SUV Purchase dataset.

---

##  Project Structure

- **Part A** – Data loading, preprocessing, feature scaling, and model training  
- **Part B** – Model evaluation and performance comparison  
- **Part C** – Interview-based theoretical and coding questions  
- **Part D** – AI-assisted explanation and evaluation  

---

##  Dataset

The dataset used is the **SUV Purchase Dataset**, which contains:

- Age  
- Estimated Salary  
- Gender (optional)  
- Purchased (Target variable)  

---

##  Workflow

1. Data Loading using Pandas  
2. Data Exploration (shape, columns, missing values)  
3. Data Preprocessing (encoding, feature selection)  
4. Train-Test Split (80/20 and variations)  
5. Feature Scaling using StandardScaler  
6. Model Training using Logistic Regression  
7. Model Evaluation using:
   - Accuracy  
   - Confusion Matrix  

---

##  Results

- Logistic Regression performs well for binary classification  
- Accuracy varies slightly with different train-test splits  
- Feature scaling improves model performance  

---

##  Key Learnings

- Understanding of classification vs regression  
- Importance of feature scaling  
- Evaluation metrics beyond accuracy  
- End-to-end ML pipeline implementation
